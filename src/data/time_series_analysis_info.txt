**Анализ временного ряда включает:**

- **Описательная статистика**: среднее, медиана, стандартное отклонение, квартили
- **Частотный анализ**: выявление доминирующих частот с помощью преобразования Фурье
- **Статистические тесты**: 
    - Тест Манна-Кендалла для выявления трендов
    - ARCH-тест для проверки гетероскедастичности
- **Сглаживание**: экспоненциальное сглаживание для выявления основной тенденции
- **Анализ остатков**: исследование отклонений от линейного тренда

Анализ поможет понять структуру данных и выбрать подходящую модель для прогнозирования.
